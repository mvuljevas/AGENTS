# Chrome Extension Vanilla Shell

This template is an AGENTS shell for a future Chrome extension using vanilla
HTML, CSS, and JavaScript.

It intentionally contains no `manifest.json`, background script, content script,
popup, options page, or runtime files. Define the extension goal first, then let
AGENTS help create the correct extension files.

## Start

```bash
agents doctor
agents setup --dry-run
agents suggest --idea "Describe the Chrome extension"
```
