# Laravel React Shell

This template is an AGENTS shell for a future Laravel application with a
React-capable frontend.

It intentionally contains no Laravel, Composer, Node, Vite, Inertia, or React
runtime files. Define the application goal first, then let AGENTS help create
the correct stack files.

## Start

```bash
agents doctor
agents setup --dry-run
agents suggest --idea "Describe the Laravel application"
```
