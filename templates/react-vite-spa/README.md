# React Vite SPA Shell

This template is an AGENTS shell for a future React, Vite, SPA, or PWA project.

It intentionally contains no `package.json`, HTML, source files, Vite config, or
runtime dependencies. Define the application goal first, then let AGENTS help
create the correct project files.

## Start

```bash
agents doctor
agents setup --dry-run
agents suggest --idea "Describe the React application"
```
