# Docs Only Shell

This template is an AGENTS shell for documentation, workflow, governance, or
planning repositories.

It intentionally contains no generated project content beyond AGENTS workflow
files. Define the project goal first, then let AGENTS help populate the real
documentation structure.

## Start

```bash
agents doctor
agents setup --dry-run
agents suggest --idea "Describe the documentation project"
```
